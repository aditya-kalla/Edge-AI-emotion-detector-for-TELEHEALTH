# 👁️ EmotionSensor-SOTA: Edge-AI Emotion Detection

A high-precision, real-time emotion recognition engine built on **YOLOv10-Nano**. Optimized for telehealth applications and low-power edge chipsets (ESP32-S3/Cortex M4).

## 🚀 Project Status: Phase 4 Complete (SOTA Reached)
**Current Benchmark:** mAP@50 = **0.7857** 🏆

## 🛠️ Tech Stack
- **Model:** YOLOv10n (NMS-Free Architecture)
- **Framework:** PyTorch, Ultralytics
- **Optimization:** Nesterov SGD, Adversarial Feature Masking
- **Deployment:** Streamlit (Web), Target: INT8 Quantized Edge Hardware
